# PikpakVerificationCode

短效邮箱一键提取PikPak验证码

## 使用方法：

```python
# 设置客户端 ID 和邮箱信息
client_id = ""  # 替换为你的 client_id
email = ""  # 替换为你的邮箱
refresh_token = ""  # 替换为你的 refresh_token
```

由于微软邮箱新机制，闪邮箱提供了refresh_token为用户提供登录邮箱的服务：[邮箱批发|-闪邮箱 (shanyouxiang.com)](https://shanyouxiang.com/)

按图所示，将店铺提供的短效邮箱信息依次替换到代码变量内即可。
